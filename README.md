# Decor Decor (Polco Deco)

Decor Decor es una PWA de nicho para decorar photocards de forma virtual, con una experiencia cozy y relajante. Este repositorio incluye una demo estática del concepto y la documentación de producto.

## Ejecutar la demo local
Puedes abrir la demo con cualquier servidor estático.

```bash
python -m http.server 8000
```

Luego visita `http://localhost:8000` en tu navegador.

## Contenido
- `index.html`, `styles.css`, `app.js`: demo visual del dashboard/editor/gachapon.
- `docs/PRODUCT_VISION.md`: visión, objetivos y principios del producto.
- `docs/UX_UI_GUIDE.md`: lineamientos visuales y de UX.
- `docs/MVP_SCOPE.md`: alcance del MVP y funcionalidades core.
- `docs/USER_STORIES.md`: historias de usuario prioritarias.
- `docs/DATA_MODEL.md`: propuesta de modelo de datos en Firebase.
- `docs/ARCHITECTURE.md`: arquitectura sugerida del stack.
- `docs/ROADMAP.md`: roadmap y fases de entrega.
- `docs/IMPLEMENTATION_PLAN.md`: plan de implementación por etapas.

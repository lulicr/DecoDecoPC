# Historias de Usuario (MVP)

## Autenticación
- Como usuario nuevo, quiero registrarme con email o Google para guardar mis proyectos.
- Como usuario recurrente, quiero iniciar sesión rápidamente para continuar mis diseños.

## Editor
- Como usuario, quiero subir una foto para usarla como base de la photocard.
- Como usuario, quiero arrastrar/rotar/escalar stickers para decorar libremente.
- Como usuario, quiero guardar proyectos a medio terminar y volver más tarde.

## Inventario y Gachapon
- Como usuario, quiero abrir cápsulas y recibir stickers por rareza.
- Como usuario, quiero ver todos mis stickers en un inventario.

## Economía
- Como usuario, quiero ganar monedas por iniciar sesión a diario.
- Como usuario, quiero gastar monedas para abrir cápsulas.

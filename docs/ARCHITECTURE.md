# Arquitectura Sugerida

## Frontend
- HTML5 + Tailwind CSS.
- JavaScript ES6+.
- Fabric.js para canvas y manipulación de elementos.

## Backend
- Firebase Authentication (email/password y Google).
- Firestore para proyectos, inventario y stats.
- Cloud Storage para imágenes y assets.

## Despliegue
- Vercel o Netlify para frontend.
- Firebase Hosting como alternativa para PWA.

## PWA
- Service Worker para caché de assets.
- Instalación en móviles.

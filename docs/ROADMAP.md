# Roadmap

## Fase 1: Fundación (2-3 semanas)
- Setup del repositorio y stack base.
- Diseño UI inicial y tokens de estilo.
- Configuración de Firebase (Auth, Firestore, Storage).

## Fase 2: Editor MVP (3-4 semanas)
- Canvas 85:55 con Fabric.js.
- Carga de imagen base.
- Manipulación de elementos.
- Guardado de proyectos (draft/final).

## Fase 3: Gachapon + Inventario (2-3 semanas)
- Flujo de apertura de cápsulas.
- Sistema de rarezas y probabilidades.
- Inventario del usuario.

## Fase 4: Economía + UX polishing (2 semanas)
- Daily rewards y misiones simples.
- Ajustes de UI y animaciones suaves.
- Optimización PWA.

# UX & UI Guide

## Estética general
- **Estilo:** minimalista cozy.
- **Bordes:** 25px (curvas suaves).
- **Sombras:** soft depth (difusas y profundas).

## Paleta base
- **Neutros cálidos:** crema, arena, café con leche.
- **Acentos:** verde musgo, terracota suave.

## Tipografía sugerida
- Sans redondeada para títulos.
- Sans legible para cuerpo (peso 400/500).

## Layout
### Dashboard
- Vista limpia con acceso inmediato a inventario, tienda y editor.
- Tarjetas modulares para acciones rápidas.

### Editor
- “Mesa de trabajo” que simula un escritorio real.
- Canvas 85:55 centrado con controles laterales.
- Barra de herramientas simple (arrastrar, rotar, escalar, duplicar, eliminar).

### Tienda / Gachapon
- Animaciones suaves y no intrusivas.
- Muestra clara de rarezas y probabilidades.

## Componentes clave
- **Botón primario:** relleno sólido, shadow soft, radio 25px.
- **Tarjeta:** borde suave + sombra profunda.
- **Modales:** fondo crema, esquinas redondeadas.

## Accesibilidad
- Contraste mínimo AA en textos.
- Tamaño de texto base 16px.

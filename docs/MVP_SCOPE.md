# Alcance del MVP

## A. Usuarios y Persistencia
- Registro / inicio de sesión (Email/Password + Google).
- Cloud Save de proyectos en progreso y finalizados.
- Perfil con estadísticas básicas:
  - Stickers coleccionados.
  - Decor Coins acumuladas.

## B. Editor de Photocards
- Canvas interactivo con proporción 85:55.
- Manipulación de objetos: arrastrar, rotar, escalar, duplicar, eliminar.
- Carga de foto base.
- Filtros con capas de textura: plástico, holograma, brillo.

## C. Gachapon y Gamificación
- Apertura de sobres/cápsulas con animaciones suaves.
- Rarezas:
  - Común (70%)
  - Raro (20%)
  - Ultra Raro (10%)
- Inventario de ítems obtenidos.

## D. Economía
- Decor Coins (moneda virtual).
- Recompensa diaria: +100 coins.
- Misiones simples (p. ej. completar decoraciones).
